    # Sequencing

    power supplies are sequenced (Vccint + Vccbram) -> (Vccaux) -> (Vcco)

    1V -> 1.8V -> 3.3V

    5V is fed into PVIN3

    (1V comes up as 5V comes up) TRKSS1 -> softstarts on 2200pf cap TRKSS2 -> held low RUN3 -> held low RUN4 -> held low

    (1.8V comes up) TRKSS1 -> started TRKSS2 -> held low RUN3 -> pulled high as 5V comes up RUN4 -> held low

    (3.3V comes up) TRKSS1 -> started TRKSS2 -> softstarts once PG3 high RUN3 -> started RUN4 -> held low

    (1.5V comes up) TRKSS1 -> started TRKSS2 -> started RUN3 -> started RUN4 -> pulled high as 3.3V comes up
