    # Main Components

    • SODIMM 204 pin DDR3 ram stick Corsair DDR3L
    • Flash memory 32M SPI Flash SPI Memory
    • 100Mhz Clock
    • Boot SPI flash plugs in through right angle PCI connector
    • XC7A200T-1SBG484C Artix-7 FPGA
    • M.2 connection to motherboard
