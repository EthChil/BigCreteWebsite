Big Crete FPGA Development Board


  • 10Mhz Clock Speed
  • 4-Layer stackup
  • Powered and programmed over USB
  • SPI Flash for FPGA configuration
  • 10x15 LED Display

The Big Crete FPGA board is built around a low cost iCE40HX1K which requires a minimal amount of parts to operate. The other two major ICs on this board is the USB to UART chip from FTDI which allows the FPGA to be flashed over USB along with a voltage convertor IC which generates the required 3.3V and 1.2V rails from the supplied 5V. This board was programmed to run a simple game of pong fully implemented in digital logic. The full board has a BOM cost of roughly $100 CAD.

The Big Crete Photonic Display System


  • 80 RGB LEDs
  • Laser Cut Construction
  • 4 Laser cut edge lit acrylic Lixie tubes
  • Brass capacitive touch interface
  • USB-C connection
  • Realtime clock module with battery backup
  • Powered by Arduino

The Big Crete Photonic Display System is a modern take on an old design with some extra big crete twists. The first Lixie tube has 2 numbers with the following panes being emojis which have varying functions from a settings menu to a sleep mode which partially shuts the panes off. The primary funciton of the Big Crete Photonic Display System is to display time and track the amount of days since preset dates. This allows a user to track metrics such as snapchat streaks or days left in the year or week easily at a glance.
The brass capacitive touch buttons give a clean interface that won't wear down over time. The lixie panes are designed to be easily swapped should the user want to change the font or icons used. The 80 RGB LEDs are WS2812B parts with 20 LEDs per Lixie bulb. The USB-C power interface gives a modern standard that is reversible and capable of keeping up with the power the BCR-140239 demands.
